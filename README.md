# matrix

## Description

The goal of this project is to build a linear algebra library from scratch, without relying on external mathematical libraries. Throughout the project, fundamental concepts of vector spaces and matrix algebra are implemented while respecting mathematical correctness and algorithmic complexity constraints.

The official project subject can be found here:
* [Subject PDF](./en.subject.pdf)

This library covers the following linear algebra concepts:

* Vector and Matrix operations (addition, subtraction, scaling)
* Linear combination
* Linear interpolation
* Dot product
* Norms
* Cosine of the angle between vectors
* Cross product
* Matrix multiplication
* Matrix trace
* Matrix transpose
* Row-echelon form
* Determinant
* Matrix inverse
* Rank
* Complex vector spaces (bonus)

## Instructions

Run an exercise with:

```sh
make ex00
make ex01
make ex02
...
```

The Makefile automatically creates a Python virtual environment and executes the corresponding file from the `src/` directory.

Clean the project with:

```sh
make fclean
```

This removes the virtual environment and the generated projection executable.
